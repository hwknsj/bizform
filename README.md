## Installation

Clone the repository, simply run `yarn` to install dependencies, or `npm install`.

To run locally in development mode, run `yarn run dev` or `npm run dev`.

To build the application, `yarn start` or `npm start`

## Tooling

* Next.js, v10
* React.js, v17
* styled-components
* react-modal
